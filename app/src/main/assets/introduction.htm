Introduction

Welcome to MAME4droid (0.139u1)

MAME4droid (0.139u1) is developed by David Valdeita (Seleuco) as a port of the MAME 0.139 emulator by Nicola Salmoria and the MAME team. It emulates arcade games supported by this version of MAME which includes over 8000 different ROMs.

This version of MAME4droid was designed to be used with high-end Android devices because it is based on the PC MAME version which requires higher specifications than the older versions.

Even with a high end device (2020 note: Except if you own a Snapdragon 865 processor or up that emulates everything at full speed including 3d games from Saturn boards;), do not expect "modern" arcade games from the 90s and beyond to necessarily work at full speed or compatibility. Some games like Outrun and the Mortal Kombat series are badly optimized and require at least a 1.5ghz dual core device.

With over 8000 games supported, some games will run better than others; some games may not run at all. It is impossible to support such a vast number of titles, so please do not email the developer asking for support for a specific game.

If you have a lower end Android device, you will find MAME4droid (0.37b5) to be much faster - but it supports fewer, older ROMs and more games will require sound samples for a complete audio experience.

MAME4droid will do nothing without having at least one compatible ROM to emulate. Your first step should be to obtain compatible ROM files and place them in the designated ROMs folder (see Installation for more information).Official web page for news, source code & additional information MAME4droid official site.

Technical Note: This version doesn't have an UML back-end ARM dynamic recompiler, which means drivers based on high spec arcade CPUs won't be playable (even if they did run, the games would be prohibitively slow anyway).
