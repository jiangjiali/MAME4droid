Controls

These are default settings and may not always apply if you change controller mappings.

Virtual Pad

  • On-Screen controller and buttons
  • Mouse for analog control
  • Lightgun (tap screen; must enable under Options->Settings->Input->Touch Lightgun)

Game Pad

  • Buttons A,B,C,D,E,F on the controller map to buttons Button 1 to 6
  • SELECT button inserts coin/adds credit
  • START button starts 1P game

Special

  • START+UP starts 2P game. START+RIGHT starts 3P game. START+DOWN starts 4P game
  • SELECT+UP inserts 2P credits. SELECT+RIGHT inserts 3P credits. SELECT+DOWN inserts 4P credits
  • R1 + START loads a save state
  • L1 + START saves a save state
  • L2 exits to selection menu to select another game
  • START + SELECT accesses the game's MAME menu (dip switches, etc)
  • R2 opens help and settings menu

Note: To type OK when MAME requires it, press LEFT and then RIGHT.
